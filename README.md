<h3>2. Организация, утвердившая разработку программы</h3>ООО «ЮГ-ТАРА»

<h3>3. Функции:</h3>

- наличие личного кабинета пользователя (сделано)
- система отслеживания заказов пользователя, реализованная с помощью статусов заказа:
